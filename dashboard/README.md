# Weather Routing Dashboard

A premium, interactive dashboard for visualization of weather routing data, designed with a "Japanese Bento Box" aesthetic.

## Setup

1. **Install Dependencies**:
   Ensure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Dashboard**:
   Navigate to the project root and run:
   ```bash
   streamlit run dashboard/app.py
   ```

## Features

- **Bento Grid Layout**: Data is organized in a clean, grid-based structure.
- **Interactive Maps**: Visualize routes on a dark-mode Leaflet map.
- **Performance Analytics**: Track fuel consumption and speed profiles.
- **Premium UI**: distinct dark theme with 'Inter' typography.
